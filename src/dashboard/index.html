🎙 Voice & Text Chat

Browser-based voice and text — works on any phone, no app needed.

Join Channel (Mumble)
How to use
  • Enter any name, leave password blank, press Connect
  • Type in the box at the bottom — press Enter to send
  • Voice is voice-activated — just speak, no button to hold
  • To private message someone: click their name in the user list
  • The mic button mutes/unmutes you
Use Safari on Mac or iPhone — Chrome blocks microphone access for self-signed certificates.

🔗 Mesh Communication

Do you have a Meshtastic/MeshCore/LoRa radio?

📡 Meshtastic

Public LoRa mesh network — any Meshtastic device can join and exchange text messages. Ideal for neighbourhood-wide comms without infrastructure.

Open Meshtastic

🔐 MeshCore

Private tactical mesh — this hub acts as a node and repeater. You control the network: flash your own LoRa radios with MeshCore firmware to join.

MeshCore Dashboard Flash Radio to MeshCore ↗

📻 Licensed Radio

HF digital radio modes — requires an Amateur Radio licence and a compatible transceiver.

✉️ Winlink Email (Pat)

Send and receive email over radio without internet, using the global Winlink network.

Open Pat — Winlink Email

📻 JS8Call

Keyboard-to-keyboard HF messaging via weak-signal digital mode. Join the #MCOMZ net Sundays 1500Z on 7.078 MHz.

Open JS8Call (Remote Desktop)

📡 FreeDATA

High-speed HF data modem for file transfer and peer-to-peer data exchange over radio.

Open FreeDATA (Remote Desktop)
